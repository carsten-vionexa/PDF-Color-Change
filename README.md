# PDF Color Change

Python project for automated PDF colour replacement.

## Goal

PDF files in the input folder are processed and new PDF files are written to the output folder.

## Structure

- config/farbtabelle.json
- input/.gitkeep
- output/.gitkeep
- scripts/recolor_disc_pdfs.py
- requirements.txt

## Usage

Run the script with Python:

python scripts/recolor_disc_pdfs.py
